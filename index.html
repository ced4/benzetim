Soru 1

Benzetim (simülasyon) nedir? Açıklayınız.

Tanım

Benzetim (simülasyon), gerçek bir sistem ya da sürecin bilgisayar ortamında matematiksel/mantıksal bir modelle taklit edilmesidir. Amaç, sistemi doğrudan değiştirmeden ya da maliyetli denemeler yapmadan onun davranışını anlamak, analiz etmek ve tahmin etmektir.

Benzetim = Gerçek sistemin bilgisayarda taklit edilmesi

Temel özellikleri:

Örnek: Hastane acil servisinde kaç doktor gerektiğini, gerçekte doktor işe almadan simülasyonla test edebiliriz.
Soru 2

Kesikli benzetim ve sürekli benzetim nedir? Örneklerle açıklayınız.

Kesikli Olay Benzetimi (Discrete Event Simulation)

Sistemin durumu yalnızca belirli olaylar gerçekleştiğinde değişir. Olaylar arasında sistem sabit kalır, zaman olaydan olaya atlar.

Müşteri geldi → durum değişti
Servis bitti → durum değişti
Bu iki olay arasında → hiçbir şey değişmez
Kullanım Alanı
Banka / market kuyruğu
Çağrı merkezi (telefon gelişleri)
Fabrika üretim hattı
Liman yük taşıma
Sürekli Benzetim (Continuous Simulation)

Sistemin durumu zamana bağlı olarak sürekli değişir. Diferansiyel denklemlerle modellenir, zaman küçük adımlarla (Δt) ilerletilir.

Her Δt anında hız, konum, sıcaklık... sürekli yeniden hesaplanır
Kullanım Alanı
Hava durumu tahmin modeli
Uçuş simülatörü
Kimyasal reaksiyon süreçleri
Elektrik devresi analizi
Karşılaştırma
ÖzellikKesikliSürekli
Durum değişimiOlaylarla, anlıkZamanla, kesintisiz
Zaman ilerletmeOlaydan olaya atlarKüçük Δt adımlarıyla
Matematiksel yapıOlay listesi, kuyrukDiferansiyel denklem
ÖrnekBanka kuyruğuUçuş simülatörü
Hibrit benzetim: Bazı sistemler ikisini birden kullanır. Fabrikada makine arızası kesikli, sıcaklık değişimi sürekli modellenebilir.
Soru 3

Fenerbahçe Şampiyonluk Simülasyonu

Fenerbahçe 12 yıldır şampiyon olamamaktadır. Önümüzdeki yıl şampiyon olma olasılığı %30'dur. Şampiyon olamadığı her yıl bu olasılık %2 daha azalmaktadır. Kaç yıl sonra şampiyon olabileceğini simüle eden Python programı yazınız.

🎯 Senin Son Hanenine Göre Değerler
Sol menüden okul numaranın son hanesini gir.
ParametreSenin Değerin
Şampiyonsuzluk süresi yıl
Başlangıç olasılığı
Her yıl azalma miktarı

Python Kodu — Senin Değerlerinle
import random

def fenerbahce_simulasyon():
    olasilik = 0.30   # başlangıç olasılığı %30
    yil = 0

    while olasilik > 0:
        yil += 1
        rassal = random.random()

        if rassal < olasilik:
            print(f"{yil}. yıl: ŞAMPİYON! (Olasılık: %{olasilik*100:.0f})")
            return yil
        else:
            print(f"{yil}. yıl: Şampiyon olamadı. (Olasılık: %{olasilik*100:.0f})")
            olasilik -= 0.02   # her başarısız yılda azalır

    print("Fenerbahçe hiç şampiyon olamadı.")
    return None

Not: random.random() her çalıştırmada farklı sonuç verir — bu simülasyonun stokastik doğasıdır.

Çift Zar Simülasyonu

İki zar aynı anda atılmaktadır. Kaçıncı denemede her iki zarın da 6 geldiğini simüle eden Python programı yazınız.

Python Kodu — Senin Değerlerinle
import random

def zar_simulasyon():
    deneme = 0

    while True:
        deneme += 1
        zar1 = random.randint(1, 6)
        zar2 = random.randint(1, 6)

        print(f"Deneme {deneme}: Zar1={zar1}, Zar2={zar2}")

        if zar1 == 6 and zar2 == 6:
            print(f"\n>>> {deneme}. denemede hedef geldi!")
            return deneme
